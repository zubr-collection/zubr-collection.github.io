Доступность

Заявление о доступности для ZUBRY.BY

ZUBRY.BY стремится обеспечить цифровую доступность для людей с ограниченными возможностями. Мы постоянно улучшаем пользовательский опыт для всех и применяем соответствующие стандарты доступности.

Меры по поддержке доступности

ZUBRY.BY принимает следующие меры для обеспечения доступности ZUBRY.BY:

  • Включить доступность в общую миссию.
  • Использовать формальные методы обеспечения качества для доступности.

Статус соответствия

Web Content Accessibility Guidelines (WCAG) определяет требования для дизайнеров и разработчиков, чтобы улучшить доступность для людей с ограниченными возможностями. Он определяет три уровня соответствия: Level A, Level AA, and Level AAA.

ZUBRY.BY частично соответствует WCAG 2.1 level AA. Частичное соответствие означает что некоторые части контента не полностью соответствуют стандарту доступности.

Доступность ZUBRY.BY полагается на следующие технологии для работы с определенной комбинацией веб-браузера и любых вспомогательных технологий или плагинов, установленных на вашем компьютере:

Эти технологии используются для соответствия используемым стандартам доступности.
